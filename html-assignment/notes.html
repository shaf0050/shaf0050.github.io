Header Tag
The "header" tag is used for the header of a page or a section. It normally contains a logo, search, navigational links, and so on.
Section Tag
The "section" tag defines a section in a document.
Lists
Lists are used to group related types of information so they are connected and simple to understand.
3 types of lists:
Unordered list - used to list related items without any particular order (ul).
Ordered list - used to list related items with a particular order (ol)
Description list - used to show name or value pairs like terms and definitions(dl)
img Tag
"img"is used to add an image in an HTML page. Adding "src" will specify the path to the image and "alt" to specify an alternate text for the image that can help the end user if the image is missing or unable to load.
Form Tag
The "form" tag is used to create an HTML form for user input.Commonly used tags for inputs are label, input, select, textarea, checkbox and radio.
